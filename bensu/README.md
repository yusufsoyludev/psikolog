# Bensu Donat Frontend

Bu proje, mevcut Stitch HTML/CSS tasarımlarını React + Vite tabanlı bir frontend uygulamasına dönüştürür.

## Başlatma

1. `npm install`
2. `npm run dev`

## Üretim için

- `npm run build`

## Özellikler

- React + Vite uygulaması
- React Router ile `/`, `/terapiler`, `/atolyeler`, `/iletisim` rotaları
- `premium_intro_animation` geçişi için intro ekranı
- `sessionStorage` ile intro bir kez gösterilir
- Global tema, font ve renkler `serene_bloom` notlarına uygun şekilde ayarlandı
- Header, Footer, Home, Therapies, Workshops ve Contact bileşenleri
- Form gönderimi frontend olarak çalışır; backend bağlantısı yok
- Harita alanı placeholder olarak bırakıldı
- Social linkleri placeholder `#` olarak ayarlandı
