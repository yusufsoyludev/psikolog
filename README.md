# Bensu Donat - Klinik Psikolog

Premium psychology clinic website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Premium Intro Animation**: Watercolor floral bloom with brand motto
- **Responsive Design**: Mobile, tablet, and desktop optimized
- **Smooth Animations**: Framer Motion scroll-triggered animations
- **Multi-page Structure**: Home, About, Therapies, Workshops, Contact
- **Premium Design**: Editorial aesthetic with botanical elegance
- **Turkish Content**: All content in Turkish, no placeholders

## Tech Stack

- React 18
- React Router 6
- Tailwind CSS 3
- Framer Motion
- Lucide React Icons
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
